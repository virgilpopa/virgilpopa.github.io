Tic Tac Toe

Game

This is a simple game of Tic Tac Toe. For this project, I have used HTML/CSS, but mostly, all the functions of the game have been coded using JavaScript.

The game is played on a 3x3 grid by two players, who alternately place the marks X and O in one of the nine spaces in the grid.

The first player to get 3 of their marks in a row (up, down, across, or diagonally) is the winner.

When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.

The functions implemented in this game allow the participants to play the game until one of the player wins 5 rounds or until the Reset button is pressed.

When a round is won or ends in a draw, the players have to press the button New Round to keep playing or they can decide to start a new game. If the Reset button is pressed, the scores are reset to 0 and the players can start a new game.
